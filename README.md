# Simple Interest Calculator

## Description
This project is a Bash script that calculates simple interest based on user input.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How it Works
The script asks the user to input:
- Principal amount
- Rate of interest
- Time period

It then calculates the simple interest using the formula and displays the result.

## Usage
Run the script using:
bash simple-interest.sh

Enter the values when prompted.

## Example
Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
